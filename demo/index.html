Client side Vector Search using EmbeddingGemma Model via LiteRT.js with Web AI

Enter some text in the box below within the Knowledge Base Management panel, seperated by paragraph line breaks, and click the store in Vector DB button. Wait until storing is complete - you only need to do this once as it is cached in IndexDB for future use even after page reload. You can then expand the Search Vector DB panel, and make a query to find relevant chunks of text that match that you could provide to a small LLM for extra context to answer complex questions about content it may not have been trained on without retraining the model (RAG)!
